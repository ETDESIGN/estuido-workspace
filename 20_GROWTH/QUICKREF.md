## 🚦 The $0 Rule

**BEFORE using a paid model, ask:**

1. ✅ Will customers see this? (External = maybe polish)
2. ✅ Is this high-stakes? (Investor/PR = polish)
3. ❌ Is this internal only? (Keep it free)
4. ❌ Is this a draft? (Keep it free)

**Default = FREE. Polish = Exception.**

## 💡 Pro Tips

1. **Chunk long content** - Split 5000-word posts into 3 MiniMax calls
2. **Batch social** - Generate 20 tweets in one Gemini call
3. **Automate first** - If you do it twice, KiloCode it
4. **Draft in free** - Only polish the winners
5. **Track costs** - Log every paid call in `costs/`
