### Phase 1: RESEARCH (Free Models Only)
```

2. Competitor analysis → Gemini Flash (fast, cheap iterations)
3. Content ideation → Gemini Flash (brainstorming at scale)
4. Data processing → KiloCode CLI (automation scripts)
```

### Phase 2: DRAFT (Still Free)
```
5. First draft → MiniMax M2.5 (long-form content)
6. Outline/structure → Gemini Flash (quick iterations)
7. SEO optimization → Gemini Flash (keyword research)
```

### Phase 3: POLISH (Paid Only When Worth It)
```
8. Final review → Claude 3.5 Sonnet (ONLY for external-facing content)
9. Critical campaigns → GPT-4o (high-stakes launches)
```

**Rule:** If it doesn't leave the building, it doesn't need polish.
