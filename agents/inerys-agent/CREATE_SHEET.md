# 🔧 Quick: Create Google Sheet

**Time**: 30 seconds

## Steps:

1. **Open**: https://sheets.google.com (logged in as inerys.contact@gmail.com)

2. **Click**: "Blank" spreadsheet

3. **Name it**: "Inerys Leads CRM"

4. **Add headers** in Row 1:
   - A1: Company
   - B1: Contact Name
   - C1: Email
   - D1: Niche
   - E1: Status
   - F1: Warm/Cold
   - G1: First Contact
   - H1: Last Contact
   - I1: Notes

5. **Copy the Sheet ID from URL**:
   - URL looks like: `https://docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit`
   - Copy the part between `/d/` and `/edit`

6. **Paste the Sheet ID here**

That's it! 🎯
