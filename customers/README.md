## Processes

### Adding New Customers
1. Create customer directory: `customers/customer-name/`
2. Create subdirectories: `{projects, tech-stack, notes, conversations, company}`
3. Generate files:
   - `README.md` - Overview
   - `CUSTOMER_PROFILE.md` - Basic info
   - `COMPANY_RESEARCH.md` - Business analysis
   - `PROJECTS.md` - Project tracking
   - `TECH_STACK.md` - Technical details
   - `NOTES.md` - Ongoing notes
4. Update this README with customer info
5. Schedule follow-up actions

### Updating Customer Info
1. Add conversation logs to `conversations/CONV_YYYY-MM-DD.md`
2. Update notes in `notes/NOTES.md`
3. Track projects in `projects/PROJECTS.md`
4. Review and update monthly
