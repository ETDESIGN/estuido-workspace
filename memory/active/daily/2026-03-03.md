### Mission Control Connection Issue (Mar 3, 2026)

**Root Cause:** OpenClaw 2026.3.x security update requires Ed25519 device identity challenge-response for WebSocket connections (mode: 'ui'). Mission Control doesn't implement this yet.

**Details:**
- Error: `code=1008 reason=device identity required`
- OpenClaw has config flag `dangerouslyDisableDeviceAuth: true` but it's broken (Issue #11590)
- Mission Control builderz-labs/mission-control doesn't support connect.challenge event

**Resolution:** Wait for next dashboard update (couple days). Don't spend time debugging.

**Reminder:** Always confirm with user before assuming free tier status!
