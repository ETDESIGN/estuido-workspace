# Omni-Hub Daily Executive Briefing Configuration

## Overview
Automated daily briefing system for Omni-Hub operations at 08:00 AM daily.

## Schedule
- **Time:** 08:00 AM (Hong Kong Time, GMT+8)
- **Frequency:** Daily
- **Delivery:** WhatsApp to E's number (+86 185 6657 0937)

## Metrics Collected

### 1. System Health
- Active sessions count
- API budget usage ($5/day limit)
- System errors and alerts
- RAM usage (5GB limit alert at 4GB)
- Disk space status

### 2. Client Pipelines
- Inerys-Agent pipeline status
- Last successful run
- Any errors or blockages
- Lead processing statistics

### 3. Team Operations
- Omni-Hub agent status
- Task completion summary
- Any blockers or escalations needed

## Alert Thresholds

### Budget Alerts
- 🟢 Normal: < $4.00 (80%)
- 🟡 Warning: $4.00 - $4.50 (80-90%)
- 🔴 Critical: > $4.50 (90%+)

### RAM Alerts
- 🟢 Normal: < 4 GB
- 🟡 Warning: 4 - 4.5 GB
- 🔴 Critical: > 4.5 GB

### Pipeline Alerts
- 🟢 Healthy: Last run < 1 hour ago, no errors
- 🟡 Lagging: Last run 1-2 hours ago, minor errors
- 🔴 Critical: Last run > 2 hours ago or critical errors
