## Quick Guide for Agents

**How to use this template:**
1. Copy this file to `requests/TOOL-REQUEST-[your-task].md`
2. Fill in your details
3. Wait for approval (<1 hour for standard tools)
4. Check back for `-DIAGNOSIS.md` and `-APPROVED.md` files

**Standard tools** (usually auto-approved):
- read, write, edit (all agents)
- exec, sessions_list (CTO only)
- browser (QA/CTO)

**Special approval needed:**
- process (kill/restart)
- gateway config
- new skill installation
