# Discord Slash Commands

Custom Discord slash commands for session management and model switching.

## Commands

### /new
Create a new session.

### /model-glm47
Switch to GLM 4.7 model (zai/glm-4.7).

### /model-kimi
Switch to Kimi K2.5 via OpenRouter (openrouter/moonshotai/kimi-k2.5).

### /model-minimax
Switch to MiniMax M2.5 via OpenRouter (openrouter/minimax/minimax-m2.5).

### /flush-context
Clear the context window for the current session.
