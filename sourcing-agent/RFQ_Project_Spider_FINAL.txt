==================================================
2. PRODUCT SPECIFICATIONS
==================================================

2.1 DOCK (Charging Station)
--------------------------------------------------

Dimensions:
- Width: 135mm
- Height: 200mm
- Depth: 200mm

Capacity:
- 10-12 batteries per dock (flexible configuration)
- Spring-loaded insertion system

Key Features:
• Pogo pin charging system (12 positions)
• Reinforced mount design
• Battery presence detection switches
• Sound emitter when battery inserted
• LED indicators for each slot
• Metal security latches (prevents forced removal)
• UART communication interface

Power:
- AC adapter input
- Distributes power to all battery slots
- Charging time: 1-1.5 hours per battery

Materials:
- Housing: ABS plastic
- Screen printing: 1-2 colors
- Metal latches: Stainless steel


2.2 BATTERY (Removable Power Pack)
--------------------------------------------------

Battery Specifications:
- Capacity: 5,000 mAh (±2000)
- Type: Lithium-ion (18650 cells, 4-cell)
- Charging: Via pogo pins ONLY (no USB charging port)

Output:
- 3 Integrated Cables:
  * Micro USB
  * USB-C
  * Lightning
- Voltage: 5V
- Current: 1A (no fast charging)
- Always ON (no power button)

Features:
• Unique serial number per battery
• LED indicator (battery level)
• Metal latch mechanism
• Redesigned cable channels


2.3 COMMUNICATION PROTOCOL
--------------------------------------------------

UART Interface Requirements:

Data to retrieve from each battery:
1. Serial number (unique ID)
2. Battery level (%)
3. Number of charge cycles
4. Battery release events
5. Battery presence status

Interface: UART to computer system

5. ISSUES TO FIX (from current product)
==================================================

Issue 1: Pogo Pin Contact
- Current: Poor contact, requires pressing
- Required: Reinforced pogo pin mount

Issue 2: Security Latches
- Current: Plastic latches (can be forced)
- Required: Metal latches (prevent forced removal)

Issue 3: Cable Management
- Current: Cables block battery insertion
- Required: Larger cable channels

Issue 4: Spring Tension
- Current: Incorrect tension
- Required: Proper tension (battery stays when horizontal)
