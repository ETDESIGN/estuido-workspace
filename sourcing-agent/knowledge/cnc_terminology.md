# CNC Machining Terminology (English ↔ Chinese)

## Materials / 材料

| English | Chinese | Notes |
|---------|---------|-------|
| Aluminum 6061 | 6061铝合金 | Most common, good machinability |
| Aluminum 7075 | 7075铝合金 | High strength, aerospace grade |
| Stainless Steel 304 | 304不锈钢 | Common, corrosion resistant |
| Stainless Steel 316 | 316不锈钢 | Marine grade, better corrosion resistance |
| Brass | 黄铜 | Easy to machine, decorative |
| Titanium | 钛合金 | Lightweight, strong, expensive |
| Plastic (POM/PEEK) | 工程塑料 | POM (聚甲醛), PEEK (聚醚醚酮) |

## Processes / 工艺

| English | Chinese | Notes |
|---------|---------|-------|
| CNC Milling | 数控铣削 | 3-axis, 4-axis, 5-axis |
| CNC Turning | 数控车削 | Lathe work, cylindrical parts |
| Surface Grinding | 平面磨削 | High precision, flat surfaces |
| Wire EDM | 线切割 | Electric discharge, complex shapes |
| Laser Cutting | 激光切割 | Sheet metal, flat patterns |

## Tolerances / 公差

| English | Chinese | Typical Value |
|---------|---------|---------------|
| Standard tolerance | 标准公差 | ±0.13mm (±0.005") |
| Precision tolerance | 精密公差 | ±0.05mm (±0.002") |
| High precision | 高精度 | ±0.01mm (±0.0005") |
| GD&T | 几何尺寸与公差 | Geometric Dimensioning & Tolerancing |

| English | Chinese | Ra Value |
|---------|---------|----------|
| As-machined | 机加工原面 | 3.2 μm (125 RA) |
| Polished | 抛光 | 0.8 μm (32 RA) |
| Anodized (Type II) | 阳极氧化（二型） | - |
| Anodized (Type III - Hardcoat) | 硬质阳极氧化 | - |
| Powder coat | 粉末喷涂 | - |
| Chrome plating | 镀铬 | - |
| Passivation | 钝化 | For stainless steel |

## Technical Terms / 技术术语

| English | Chinese | Notes |
|---------|---------|-------|
| 3-axis / 4-axis / 5-axis | 三轴 / 四轴 / 五轴 | Machine capabilities |
| Tooling | 刀具/工装 | Cutting tools or fixtures |
| Setup time | 换线时间 | Machine preparation |
| Cycle time | 加工周期 | Time per part |
| Lead time | 交货期 | Time from order to delivery |
| MOQ (Minimum Order Quantity) | 最小订货量 | Smallest order accepted |
| Prototype | 原型/样件 | First production run |
| Pilot production | 试产 | Pre-production batch |
| Mass production | 量产 | Full-scale production |
| Tolerance stack-up | 公差叠加 | Cumulative tolerance errors |
| Design for Manufacturing (DFM) | 面向制造的设计 | Optimize for production |
| First Article Inspection (FAI) | 首件检验 | Validate first production part |

## Common CNC Defects / 常见缺陷

| English | Chinese | Cause |
|---------|---------|-------|
| Tool marks | 刀痕 | Improper cutting parameters |
| Surface roughness | 表面粗糙 | Worn tools, wrong speed |
| Dimensional deviation | 尺寸偏差 | Tool wear, thermal expansion |
| Burr | 毛刺 | Sharp edges, needs deburring |
| Chatter marks | 振刀纹 | Vibration during cutting |

## Quality Inspection Terms / 质检术语

| English | Chinese | Notes |
|---------|---------|-------|
| CMM (Coordinate Measuring Machine) | 三坐标测量机 | Precision measurement |
| Caliper | 卡尺 | Basic measurement |
| Micrometer | 千分尺 | Precise measurement |
| Surface profilometer | 表面粗糙度仪 | Measure Ra value |
| Visual inspection | 目视检查 | Visual defects |
| FAI (First Article Inspection) | 首件检验 | Validate first part |
| IPQC (In-Process Quality Control) | 过程质检 | During production |
| OQC (Outgoing Quality Control) | 出货质检 | Before shipment |

## Common RFQ Questions / 常见询价问题

1. **What's your minimum order quantity?**  
   你们的最小订货量是多少？

2. **Can you provide material certificates?**  
   你们能提供材质证明吗？

3. **What's your typical lead time?**  
   你们的常规交货期是多长？

4. **Do you have ISO 9001 certification?**  
   你们有ISO 9001认证吗？

5. **Can you work with provided CAD files?**  
   你们能用提供的CAD文件加工吗？

6. **What's your tolerance capability?**  
   你们的公差能力是多少？

7. **Do you offer design for manufacturing (DFM) feedback?**  
   你们提供DFM建议吗？

8. **Can you provide first article inspection?**  
   你们能提供首件检验报告吗？
