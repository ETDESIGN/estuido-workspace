# E-Studio Sourcing Dashboard — Stitch UI Generation Prompts

## Setup
1. Go to https://stitch.withgoogle.com
2. Create a new project: "E-Studio Dashboard"
3. Set design system: Dark theme, Inter font, primary blue #3B82F6
4. Copy/paste each prompt below as a new screen
